Product Description

Data is often stored in flat files and/or relational databases. Accessing data stored in XML files requires specific coding, and is usually expensive, time consuming and could even lead to a change of the current system or software being used.

Oracle Data Integrator Driver for XML is the world's first type-4 (all Java) JDBC driver for XML files. Unlike other middleware products which transfer data between an XML document and a database, OracleDI Driver for XML treats an XML document as a JDBC data source. This allows existing JDBC applications to use XML documents as data sources.

With Oracle Data Integrator Driver for XML, you can query hierarchical and tabular XML documents using standard SQL syntax. You will also be able to save the changes made for both hierarchical and tabular formatted XML files. All changes occur within transactions that can be committed or rolled back.

Features & Benefits

XML is widely acclaimed as the data format of the Web that brings together data from diverse sources into a uniform format. With OracleDI Driver for XML, your Java applications will be able to manipulate complex hierarchical or simple tabular formatted XML files using standard ISO SQL syntax. You will be able to query your XML data using ISO SQL commands (joins, select, update, insert, delete). You will also be able to persist the changes made for both hierarchical and tabular formatted XML files.

OracleDI Driver for XML supports:

  • Tabular and Hierarchical XML file structures
  • Standard SQL (Structured Query Language) Syntax
  • Correlated subqueries, inner and outer joins
  • ORDER BY and GROUP BY
  • COUNT, SUM, MIN, MAX, AVG and other functions
  • All Standard SQL functions
  • Transaction Management
  • Referential Integrity (foreign keys)
  • Saving Changes made on XML data into the XML files

With OracleDI Driver for XML, you will:

  • easily access your XML data,
  • access your XML files as if they are forming a relational database,
  • spend no time on training for XML & XSLT languages,
  • keep your existing systems without changes.
