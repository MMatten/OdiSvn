The Assignments

All the following assignment forms are valid:

x = v

x1 = x2 = v

x1, x2 = v1, v2

x1, x2, ..., xn = v1, v2, ..., vn

(x1, x2, ..., xn) = (v1, v2, ..., vn)

[x1, x2, ..., xn] = [v1, v2, ..., vn]

 

The following special assignment forms are also valid:

x += y is equivalent to x = x + y

x *= y is equivalent to x = x * y

x /= y is equivalent to x = x / y

x -= y is equivalent to x = x - y

x %= y is equivalent to x = x % y

x &= y is equivalent to x = x & y

x ^= y is equivalent to x = x ^ y

x **= y is equivalent to x = x ** y

x |= y is equivalent to x = x | y
