OdiPingAgent

Usage

OdiPingAgent -PHYSICAL_AGENT_NAME=<physical_agent>

Description

Performs a connection test on an agent. If the agent is not started, this command generates an error.

Parameters

Parameters

Mandatory

Description

-PHYSICAL_AGENT_NAME=<physical_agent>

Yes

Name of the physical agent to test.

Note: the old syntax with AGENT_NAME is still valid but deprecated.

Examples

Test the physical agent AGENT_SOLARIS_DEV

OdiPingAgent -PHYSICAL_AGENT_NAME=AGENT_SOLARIS_DEV
