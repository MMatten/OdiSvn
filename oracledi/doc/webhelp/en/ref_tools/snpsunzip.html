OdiUnZip

Usage

OdiUnZip -FILE=<file> -TODIR=<dest_dir> [-OVERWRITE=<yes|no>] [-ENCODING=<file_name_encoding>]

Description

Used to unzip an archive file to to a directory.

Parameters

Parameters

Mandatory

Description

-FILE=<file>

Yes

Full path to the ZIP file to unzip

-TODIR=<dest_file>

Yes

Destination directory or folder

-OVERWRITE=<yes|no>

No

Indicates if the files that already exist in the target directory must be overwritten.

-ENCODING=<file_name_encoding>

No

Character encoding used for filenames inside the archive file. For a list of possible values see http://java.sun.com/products/jdk/1.2/docs/guide/internat/encoding.doc.html. Defaults to the platform's default character encoding.

Examples

Unzip the file archive_001.zip from directory C:\archive\ into directory C:\TEMP:

OdiUnZip "-FILE=C:\archive\archive_001.zip" -TODIR=C:\TEMP\
