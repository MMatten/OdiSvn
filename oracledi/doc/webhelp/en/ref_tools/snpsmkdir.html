OdiMkDir

Usage

OdiMkDir -DIR=<dir>

Description

Creates a directory.

If the parent directory does not exist, this command will recursively create the parent directories.

Parameters

Parameters

Mandatory

Description

-DIR=<dir>

Yes

Directory (or folder) to create.

Examples

Creation of the directory "odi" in C:\Program Files

OdiMkDir "-DIR=C:\Program Files\odi"
