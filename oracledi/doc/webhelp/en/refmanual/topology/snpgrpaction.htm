Action Groups

Actions (Templates for DDL commands) are organized into Action Groups. An action group corresponds to a given syntax and/or purpose (Oracle, SQL-92, etc).

Definition

Properties

Description

Action Group Name

Name of the action group, as it appears in the graphical interface.

Group Code

The group code allows the action group to be referenced among the different repositories.

Caution: Avoid changing this code, as this could make some objects inconsistent.

Default Group

Check the Default Group box if you want this action group to be selected for new models created for this technology. If no default action group is selected for a given technology, new models use the generic actions .
