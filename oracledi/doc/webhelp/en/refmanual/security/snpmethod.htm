Method

A method is a type of action which can be performed on an object. Each object has a series of methods that are specific to it. The notion of method in Oracle Data Integrator is similar to that of Object-Oriented programming.

Definition

Property

Description

Name

Label of the method that appears in the graphical interface.

Internal Java name

Name of the Oracle Data Integrator internal Java method implemented on this object.

Object creation method

Shows that this method contributes to creating an object instance

Default method granted for the object creator

If this box is checked, the object creator is automatically entitled to the method.

Note: This checkbox takes effect only for objects for which authorizations by object instance are supported.

Object Concerned

Internal id code of the object
