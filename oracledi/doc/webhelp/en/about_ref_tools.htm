Preface

Oracle Data Integrator is a unified, comprehensive product line, that enables enterprises to optimize the development and execution of all integration processes through a consistent approach, independently of latency and persistence requirements.

This manual provides a reference of the Oracle Data Integrator Tools. It is intended for application developers who want to use these tools to design integration scenarios.

Organization of This Manual

This manual contains the following:

  • Chapter 1 - Using the Oracle Data Integrator Tools explains how to use Oracle Data Integrator Tools when designing integration scenarios with Oracle Data Integrator.

  • Chapter 2 - Using Open Tools explains how to install and use tools developed by third party providers, or your own tools.

  • Chapter 3 Developing Open Tools explains how to create your own open tools.

  • Chapter 4 - Oracle Data Integrator Tools Reference provides a detailed syntax for each tool.

Oracle Data Integrator Documentation

Oracle Data Integrator documentation also includes the following:

  • Oracle Data Integrator Installation Guide: This guide contains instruction for installing and upgrading Oracle Data Integrator on any platform.

  • Getting Started with an ETL Project: This guide contains a tutorial to design Extraction Transformation & Loading (ETL) data integration scenarios.

  • Oracle Data Integrator User's Guide: This guides describes how to work with Oracle Data Integrator graphical user interface. It guides you through procedures and examples to manage development in Data Integrator.

  • Oracle Data Integrator Reference Manual: This manual describes the details of Data Integrator graphical user interface.

  • Oracle Data Integrator Tools Reference Manual: This manual provides a reference for the Data Integrator Tools.

  • Oracle Data Integrator Substitution Methods Reference: This manual provides a reference of the Data Integrator Substitution Methods.

  • Jython Quick Reference: This manual provides a reference of the Jython scripting language.

These guides are available in FlashHelp (For reading from a web browser) and PDF format. Online help is also available within the Data Integrator modules graphical user interface as a multi-platform JavaHelp.
