Filter a View

The objects displayed in a view can be simply filtered.

At the top of the view, a input field appears.
If you start typing in this field, only items of the tree view named like your input appear in the tree. All other items are filtered.

If you click the button at the right of the field or erase your input, the filter is disabled and the filtered objects reappear.

Filtering in a view supports the following options:

  • Case sensitivity.

  • Match the input anywhere in the string or only at the beginning of the name.

  • Match leaf nodes only.

  • Hide the nodes without child nodes.

You can change these options by clicking the filter option button at the left of the input field.

Note: A filter applied to view is not persistent.
