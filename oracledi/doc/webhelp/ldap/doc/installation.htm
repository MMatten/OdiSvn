Installation

Prerequisites

To use OracleDI Driver for LDAP, you need a Java Virtual Machine 1.2.2 or higher. To check your Java Machine version, type the java -version command in a command prompt.

If needed, you can download the appropriate Java Machine at the Sun Microsystems website.

Receiving the Package

Unbundling The Downloaded File

You can unbundle the downloaded file using a tool such as WinZip, Pkunzip or jar. For example:

c:> jar xvf OracleDILdap.zip

or

c:> pkunzip -d OracleDILdap.zip

Note:  Pay attention to specifying the option that preserves path names in the compression tool. For example, specify "-d" for pkunzip

The downloaded file contains the following files:

File/Directory Name

Description

snpsldapo.jar

Oracle Data Integrator Driver for LDAP

/doc

OracleDI Driver for LDAP Documentation

/demo

Contains the demo and sample files for OracleDI Driver for LDAP

Installing the Package for Java applications

Configuring the Classpath

Once you have the driver installed, set the classpath variable by adding the jar and zip file paths.  For example:

Under Windows:

set CLASSPATH=%CLASSPATH%;c:\snpsldapo.jar

Under Unix:

setenv CLASSPATH "/usr/local/snpsldapo.jar:$CLASSPATH"

Important Note: If using an External Database engine you must also make sure that the JDBC driver used to connect the external database and the .properties file are in the classpath.

Installing the Package for Oracle Data Integrator

The Oracle Data Integrator application requires the LDAP (JDBC Edition) driver to be installed and referenced in the Java path.

To use the LDAP (JDBC Edition) driver, you must copy the driver's jar or zip file to the/drivers directory. Oracle Data Integrator will automatically detect the driver the next time it is launched.

Caution: The automatic detection of drivers does not function correctly on Windows 95 and 98. For these systems you must add the driver's filename in the PROJJDBC path parameter in the snpparams file.

In general, external parameter files such as the bundle or .properties files are placed in the Oracle Data Integrator  /bin directory..
